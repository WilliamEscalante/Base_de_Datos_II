Que es una base de datos relacionales:  

Una base de datos no relacional 
es un conjunto de datos que están almacenados en una forma no tabular


Que es DML: 

Lenguaje de Manipulación de Datos es un lenguaje proporcionado 
por los sistemas gestores de bases de datos que permite a los usuarios de 
la misma llevar a cabo las tareas de consulta o modificación de los datos 
contenidos en las Bases de Datos



que es DDL:

El lenguaje de definición de datos (DDL) es un lenguaje de computadora 
utilizado para crear y modificar la estructura de los objetos de la base
de datos en una base de datos
